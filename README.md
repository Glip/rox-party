# Rox Party Groups

Веб‑приложение для распределения игроков по группам на основе данных из Google Sheets. Работает полностью в браузере: только HTML, CSS, JavaScript и `localStorage`, без PHP и серверной части.

## Структура проекта

- `index.html` — всё приложение (разметка, стили, логика).
- `img/` — иконки профессий (картинки для карточек игроков).

## Требования

- Любой современный браузер (Chrome, Firefox, Edge, Safari).
- Для работы с Google Sheets таблица должна быть:
  - доступна **«для всех по ссылке (только просмотр)»**;
  - иметь в первой строке заголовки колонок: «Имя», «Уровень», «Профессия» (регистр не важен).

## Запуск локально

Рекомендуется запускать через простой HTTP‑сервер (иначе возможны ошибки CORS / `Failed to fetch`).

### Вариант 1: Python

```bash
cd c:\dev\rox-party
python -m http.server 8000
```

Откройте в браузере:

```text
http://localhost:8000/index.html
```

### Вариант 2: Node.js (serve)

```bash
cd c:\dev\rox-party
npm install -g serve
serve .
```

Далее откройте адрес, указанный в консоли (обычно `http://localhost:3000` или `http://localhost:5000`).

## Настройка Google Sheets

1. Создайте или откройте таблицу Google Sheets.
Пример таблицы `https://docs.google.com/spreadsheets/d/1jveA22ixffOG7TaE4Fa5EKOP3QLEALDBQjsUT21y7yU/edit?gid=0#gid=0`

<img width="554" height="752" alt="image" src="https://github.com/user-attachments/assets/9e04036a-f9bc-4b58-a305-44cae27f7189" />

2. Включите доступ: **Файл → Поделиться → Любой, у кого есть ссылка → Просмотр**.
3. Убедитесь, что в первой строке есть столбцы:
   - «Имя»
   - «Уровень»
   - «Профессия»
4. Скопируйте ссылку формата:

   ```text
   https://docs.google.com/spreadsheets/d/ID/edit#gid=GID
   ```

5. Откройте приложение, вставьте эту ссылку в поле «Ссылка на Google Sheets» и нажмите **«Загрузить»**.

## Использование приложения

1. **Загрузка списка игроков**
   - Вставьте ссылку на таблицу в верхнее поле.
   - Нажмите кнопку **«Загрузить»**.
   - Игроки появятся в колонке «Не распределены».

2. **Распределение по группам**
   - Потяните карточку игрока мышкой и бросьте её на нужную группу.
   - Внутри групп действуют ограничения по размеру (по умолчанию 5 человек).

3. **Сортировка**
   - В блоке «Сортировка» выберите тип сортировки (по имени, уровню или профессии).
   - Нажмите **«Сортировать нераспределенных»**, чтобы отсортировать только колонку «Не распределены».

4. **Сохранение и загрузка наборов**
   - В поле **«Имя нового набора»** введите имя (например, `Вечеринка_01`).
   - Нажмите **«Сохранить»** — текущая раскладка групп будет сохранена в `localStorage` браузера.
   - Чтобы загрузить сохранённый набор:
     - выберите его в выпадающем списке;
     - нажмите **«Загрузить»**.
   - Сохранения **привязаны к конкретному браузеру и устройству** (между пользователями не синхронизируются).

5. **Проверка актуальности**
   - Нажмите **«Проверить актуальность»**, чтобы:
     - удалить из наборов игроков, которых уже нет в актуальной таблице;
     - добавить новых игроков в колонку «Не распределены».

6. **Сброс**
   - Кнопка **«Сбросить распределение»** очистит все группы и вернёт всех игроков в «Не распределены».
